# PvP Framework

## 前提
- Multiverse Core
- FancyNPC

## 使い方

1. サーバーに前提プラグインもぶち込む
2. まず起動
3. /plugins/PvPFramework/config.yml のワールドなどを変更（もともとはworld）
4. /plugins/PvPFramework/game-templates のフォルダ内の作りたいゲームのymlをコピーして /plugins/PvPFramework/games/ に置く
5. /plugins/PvPFramework/games/ のymlを編集してワールドや座標などを変更
6. マイクラで /pvpf reload でリロード
7. 遊べ

詳細はこちら
https://forum.bac0n.f5.si/t/topic/1154
